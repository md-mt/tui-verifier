These artifacts were generated by the recipes in `examples/`:

```bash
uv run tui-verify run examples/pi_help.recipe.json --video
uv run tui-verify run examples/pi_version.recipe.json --video
uv run tui-verify run examples/pi_list.recipe.json --video
```

On this machine Pi exits with `71` after printing its launcher banner because
the Meta launcher cannot apply its sandbox in the current execution
environment. The recipes intentionally preserve that exit code as evidence
while asserting the observable terminal surface.
